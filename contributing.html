# Contributing to AeroPax Live Thank you for your interest in contributing to AeroPax Live! We welcome contributions from the community to help make this flight tracking app even better. ## How to Contribute ### Reporting Bugs If you find a bug, please create an issue with the following information: - **Title:** A clear and descriptive title - **Description:** A detailed description of the problem - **Steps to Reproduce:** Step-by-step instructions to reproduce the issue - **Expected Behavior:** What you expected to happen - **Actual Behavior:** What actually happened - **Environment:** - iOS version - Device model - App version - **Screenshots:** If applicable, add screenshots or screen recordings ### Suggesting Enhancements We love new ideas! To suggest an enhancement: - Check existing issues to see if it's already been suggested - Create a new issue with the "enhancement" label - Clearly describe the feature and its benefits - Provide examples or mockups if possible ## Communication - **Email:** admin@av8r.com.au for direct inquiries - **GitHub Issues:** For bug reports and feature requests - **Pull Requests:** For code contributions ## Code of Conduct ### Our Standards - Be respectful and inclusive - Welcome newcomers and help them learn - Focus on constructive feedback - Respect different viewpoints and experiences ### Unacceptable Behavior - Harassment or discriminatory language - Trolling or insulting comments - Publishing others' private information - Other conduct that's inappropriate in a professional setting ## Recognition Contributors will be recognized in our release notes and may be listed in a CONTRIBUTORS file. ## Questions?
